# AGENTS.md

Read [CLAUDE.md](./CLAUDE.md) before making changes in this repository. It is
the source of truth for build commands, architecture notes, testing guidance,
and issue-tracking workflow.

## Commit Messages

Use Conventional Commits for all commits, for example:

```text
feat(install): add prebuilt binary installers
```

Choose the type and scope that match the change (`feat`, `fix`, `docs`,
`build`, `chore`, and so on).
